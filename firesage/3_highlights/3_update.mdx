# Update

This page discusses how `update` works.

import { Image } from '@site/src/components/Image'

:::info
The implementation of `update` is largely different than official SDK's update due to [deficiency](https://stackoverflow.com/questions/73263183/typescript-type-of-key-is-string-if-key-is-template-literal-a-string) in Typescript.

Luckily array as parameters are not bad at all, personally I enjoy using tihs interface.
:::

## Type Check

RTDB `update` does not have weird quirks like like Firestore, so the type check is straight forward:

<Image
	src="https://github.com/tylim88/FirelordJSDoc/blob/main/static/firesage/update1.png?raw=true"
	text="update type checks"
/>

:::tip
`update` accept `T[]` data type if the node type is `NumericKeyRecord<T>`
:::
