# Library Exports

FireSageJS wraps and exports every function found in the `@firebase/database/dist/public.d.ts` with few exceptions.

:::note
FireSageJS does not suppport anything related to read or set priority.
:::

## Functions

These are exports with wrapping:

- increment
- serverTimestamp
- onChildAdded
- onChildChanged
- onChildMoved
- onChildRemoved
- onValue
- get
- push
- remove
- set
- update
- endAt
- endBefore
- equalTo
- limitToFirst
- limitToLast
- orderByChild
- orderByKey
- orderByValue
- startAfter
- startAt

These are exports without wrapping:

- getDatabase
- connectDatabaseEmulator
- enableLogging
- goOffline
- goOnline
- forceLongPolling
- forceWebSockets
- off
- onDisconnect
