# Contributing

1. Simplify type logic and remove dead code.

2. Implement latest Typescript features.

3. Cover more type tests and run time tests.

4. Better [documentation](https://github.com/tylim88/FirelordJSDoc).

5. Implement `mandatory field type`, a special type that mark keys as `required` in `update` operations. Practical usage is to mark a key like `updatedAt`.

6. Query Constraint type change Read type. For example for query `where('a', '==', true)`, we can further narrow down the type of field `a` to only `true`.

7. New Logo

8. Functionality test for onDisconnect.
