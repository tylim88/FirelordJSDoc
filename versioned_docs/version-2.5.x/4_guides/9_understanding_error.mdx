# Understanding Error Message

This section discusses how to understand error messages from Typescript.

## Take The Last One

import { Image } from '@site/src/components/Image'

Always looking at the **last** error message and ignore the rest. Example:

<Image src="/firelordv2.5/error1.png" text="take the last error message" />

In this example, the correct field path is `a` and `__name__`(`__name__` is a default comparison path of every document, see [document Id](./documentId))
