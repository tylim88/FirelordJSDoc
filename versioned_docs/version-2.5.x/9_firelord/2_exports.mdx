# Library Exports

Firelord wraps and exports every function found in the `firebase-admin\lib\firestore\index.d.ts`.

:::note
Firelord does not support any function and methods related to `withConverter`.

Because `Meta Type` should be the single source of truth and if we allow `withConverter`, we have the second source of truth: the returns of `toFirestore` and `fromFirestore`.
:::
