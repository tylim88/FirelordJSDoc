## WHat is Direct Nested Array ğŸ•Šï¸

Direct nested array type is array type where its element type is also an array, eg: `number[][]` and `[{a:string[][]}]`, such types are invalid in Firestore.

:::info
indirect nested array is a valid type, eg: `[{ a:string[] }]` and `[{ a:{ b:string[] }[] }]`
:::

If you accidentally defined an direct nested array type, FirelordJS replace it with an error message.
