### No Union With Object Type ğŸ§

You cannot union any type with object type, except **[`PossiblyReadAsUndefined`](./highlights/possibly_undefined.md)**.

If you do, FirelordJS will replaces it with error message:

<Image
	src="https://github.com/tylim88/FirelordJSDoc/blob/main/static/img/metaType5.png?raw=true"
	text="FirelordJS forbid you from union any type with object type."

Do not use `Record<number, unknown>` in any way, numeric index is not parseable by Typescript.
