# Minified Size

We obtained the minified size using `rollupjs` with below configuration:

```ts
import typescript from '@rollup/plugin-typescript'
import { terser } from 'rollup-plugin-terser'

export default {
	input: 'src/index.ts',
	output: {
		dir: 'dist',
		format: 'cjs',
	},
	plugins: [
		typescript({
			tsconfig: 'tsconfig.prod.json',
		}),
		terser(),
	],
}
```

You can run `npm run rollup` in the source code to build the minified `js` file.

However we do not publish the minified files, we publish the compiled files so you can:

- view the source code in node_modules.
- minify it with your choice of tool and setting.
