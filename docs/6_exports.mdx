# Library Exports

FirelordJS wraps and exports every function found in the `@firebase/firestore/dist/index.d.ts`.

## Functions

These are exports with wrapping:

- queryEqual
- refEqual
- snapshotEqual
- documentId
- arrayRemove
- arrayUnion
- deleteField
- increment
- serverTimestamp
- onSnapshot
- addDoc
- deleteDoc
- getDoc
- getDocs
- setDoc
- updateDoc
- endAt
- endBefore
- orderBy
- startAfter
- startAt
- where
- collection
- collectionGroup
- doc
- query
- runTransaction
- writeBatch

These are exports without wrapping:

- Timestamp
- GeoPoint
- Bytes
- getFirestore
- terminate
- initializeFirestore
- loadBundle
- clearIndexedDbPersistence

- enableIndexedDbPersistence

- enableNetwork
- onSnapshotsInSync
- namedQuery
- waitForPendingWrites
- CACHE_SIZE_UNLIMITED
